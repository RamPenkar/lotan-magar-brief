# Lotan Magar — Strategy Brief

One-page editorial site presenting the strategy brief + cautions for Lotan Magar.

Hebrew, RTL, scroll-reveal animations, mobile-responsive. Search engines blocked via `noindex` meta + `robots.txt`.

## Enable GitHub Pages (one-time setup)

1. Go to **Settings → Pages** in this repo.
2. Under **Source**, choose **Deploy from a branch**.
3. Branch: `main`, folder: `/ (root)`. Click **Save**.
4. Wait 30–90 seconds. Refresh the Pages settings page until the URL appears.

## Final URL

`https://rampenkar.github.io/lotan-magar-brief/`

## Sharing notes

- Anyone with the URL can view the page.
- Search engines won't find it (blocked via robots).
- For real auth-gated access, upgrade to GitHub Pro and make the repo private — Pages on private repos require a paid plan.

## Updating the content

Edit `index.html` directly on GitHub or locally. Changes pushed to `main` auto-deploy to Pages within ~1 minute.
